Instruction for installing adonis template with full PyroCMS package:

1-Login to your PHPmyadmin.
2-Create a database 
3-Import the Adonis.sql file.
4-Open Database.php file located at system/cms/config/database.php and put your username & password and database name.
5-Upload Files folder to your host server.
6-Go to http://yoursite/admin and login :username:admin & password:123456
